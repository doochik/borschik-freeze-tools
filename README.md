# Borschik freeze tools

Improved freeze tools for [Borschik](https://github.com/veged/borschik)

## Freeze images in JS
```(js)
document.write('<img src="' + borschik.freeze('1.png') + '"/>')
```

```(bash)
$ borschik --tech borschik-tech/freeze-js-imgs --input 1.js > 1.freeze.js
```
